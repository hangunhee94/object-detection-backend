## 3. 핵심 기능
>이 프로젝트의 핵심 기능은 머신러닝을 통한 얼굴이 보여지는 사진속 얼굴의 나이를 예측해주는 웹페이지입니다.   
>사용자는 각종 얼굴 사진을 등록하여 얼굴 나이를 예측 해볼 수 있습니다.
